## What is Scorely?

Scorely is a curated marketplace where you can find the perfect AI agent for any task. Whether you need a code reviewer, data analyst, or customer support bot — browse community-vetted agents, read real reviews, and pick the right tool for the job.

Think of it as an app store, but for AI agents.

## Features

- **Browse Trending Agents** — See the most popular AI agents ranked by community votes and ratings
- **Submit Your Agent** — List your own AI agent for the community to discover and review
- **Community Voting** — Upvote and downvote agents to surface the best ones
- **Agent Profiles** — View detailed info including descriptions, ratings, GitHub repos, and MCP endpoints
- **Category Filtering** — Find agents by category: Development, Analytics, Research, and more
- **Beautiful UI** — Glassmorphism design with smooth animations and dark mode support

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org) 20 or later
- A free [Supabase](https://supabase.com) account

### Quick Start

**1. Clone the repo**

```bash
git clone https://github.com/levihgibbons1/scorely.git
cd scorely
```

**2. Install dependencies**

```bash
npm install
```

**3. Set up your environment**

Create a `.env` file in the project root:

```
SUPABASE_URL=your-supabase-project-url
SUPABASE_ANON_KEY=your-supabase-anon-key
```

You can find these values in your [Supabase dashboard](https://app.supabase.com) under **Settings > API**.

**4. Start the app**

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and you're in.
